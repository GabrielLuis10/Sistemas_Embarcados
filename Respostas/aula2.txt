1. Por que o Linux recebeu esse nome?
	R: O nome Linux é uma referência ao seu criador Linus Torvals e ao sistema base Unix

2. O que são daemons?
	R: Daemons são processos que o usuário não possui controle direto uma vez que são executados em plano de fundo pelo sistema operacional

3. O que é o shell?
	R: Shell é uma interface para o usuário que o possibilita acesso aos serviços do sistema opercional. Pode ser tANTO uma interface gráfica (GUI) ou um consele por
linha de comando (CLI).

4. Por que é importante evitar executar o terminal como super-usuário?
	R: O acesso como super-usuário permite a execução de comandos e acesso a dados criticos o que pode, sobretudo para usuários inesperientes, gerar falhas no sistema

5. Qual botão do teclado completa o que o usuário escreve no terminal, de acordo com o contexto?
	R: Tab

6. Quais botões do teclado apresentam instruções escritas anteriormente?
	R: setas para cima ou para baixo
	
7. Apresente os respectivos comandos no terminal para:
	(a) Obter mais informações sobre um comando.
		R: man comando
	(b) Apresentar uma lista com os arquivos dentro de uma pasta.
		R: ls 

	

 	(c) Apresentar o caminho completo da pasta.
		R: pwd
  
	(d) Trocar de pasta.
		R:cd diretório_destino
      
	(e) Criar uma pasta.
		R: mkdir nome_pasta
      
	(f) Apagar arquivos definitivamente.
   		R: rm -r nome_arquivo
  
	(g) Apagar pastas definitivamente.
		R: rm -r nome_diretório	

	(h) Copiar arquivos.
		R: cp - atrributos origem destinod

	
	(i) Copiar pastas. 
	        R: cp -R nome_diretório destino/
  
	(j) Mover arquivos. 
		R: mv nome_arquivo diretório_destino/	

	(k) Mover pastas. 
		R: mv nome_diretório destino/	

	(l) Renomear pastas.
		R: mv -R nome_atual nome_desejado 

	(m) Apresentar o conteúdo de um arquivo.
		R: cat arquivo 

	(n) Apresentar o tipo de um arquivo. 
		R: file nome_arquivo

	(o) Limpar a tela do terminal.
		R: clear 

	(p) Encontrar ocorrências de palavras-chave em um arquivo-texto.
		R: cat arquivo_texto | grep palavra_chave
 
	(q) Ordenar informações em um arquivo-texto. 
		R: sort nome_arquivo

	(r) Substituir ocorrências de palavras-chave em um arquivo-texto.
		R: sed -e `/palavra1/palavra2` arquivo.txt
 
	(s) Conferir se dois arquivos são iguais. 
		R: cmp arquivo_1 arquivo_2

	(t) Escrever algo na tela.
		R: echo texto
